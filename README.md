# 捷报体育——体育足球篮球

  "捷报体育——体育足球篮球"是一款线上预定体育项目(足球/篮球)的应用。
  
  【核心功能】
  "捷报体育——体育足球篮球"是一款线上预定足球场馆、篮球场馆的应用。
  
  【预定说明】 
  应用内场馆(足球篮球)配备4-6号馆，用户根据需要进行选择场馆。
  
  【预定方式】 
  所有场馆均采用线上预定的方式，选择场馆区域、选择预约日期和时间、填写联系人信息。

  【资讯】
  资讯为用户提供来自国内关于足球篮球的新闻。
